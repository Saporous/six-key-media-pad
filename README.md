# six-key-pad
Sketch for macro keypad with 6 keys (2 rows, 3 columns), used for media controls

1. Vol down 
2. Mute
3. Vol up
4. Previous track
5. Play/pause
6. Next track

Hardware :

+ Pro Micro (Arduino compatible ATMega32u4) or clone
+ Cooler Master keyswitch tester or any 3x2 array of switches in an enclosure
